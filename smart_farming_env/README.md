# 🌾 Smart Farming Environment

This project simulates a real-world farming system where an AI agent can make decisions like irrigation, fertilization, and harvesting.

## 🚀 Features
- Real-world agriculture simulation
- step(), reset(), state() API
- Reward-based learning
- 3 difficulty tasks
- Scoring system (0.0 – 1.0)

## 📡 API Endpoints
- `/` → Home
- `/reset` → Reset environment
- `/state` → Get current state
- `/step/{action}` → Perform action

## 🎯 Tasks
- Easy → Improve crop health
- Medium → Balance water & health
- Hard → Efficient harvest
